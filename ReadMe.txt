This project uses linear regression to analyze previous data on the Olympics starting in the early 20th century 
and estimate how many medals said country will acquire using several columns with high correlations to medals earned.

* Code is run using: python project.py
* neccessary libraries: pandas and scikit-learn
* optional libraries (genetrates graphs): seaborn and matplot
* when countries are inputted, they must be capitalized,
	ex: United States, Canada, Croatia
